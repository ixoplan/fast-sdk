# fast-sdk
PHP SDK for the FAST - Frontend Application Server Technology(formerly CDE) APIs

## Installation

Simply add `ixoplan/fast-sdk` to your composer.json, e.g:

```json
{
    "name": "myvendor/myproject",
    "description": "Using fast-sdk",
    "require": {
        "ixoplan/fast-sdk": "*"
    }
}
```
